Introduction to HTML

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content. Along with CSS and JavaScript, HTML is a cornerstone technology used to create web pages as well as to create user interfaces for mobile and web applications.

Paragraphs

This is a paragraph. Paragraphs are block elements in HTML that are used to organize written content on the web page.

Images

Images can be inserted into your web page using the <img> tag. The 'src' attribute specifies the path to the image, and the 'alt' attribute provides alternative text for the image if it cannot be displayed.

Lists

There are two types of lists in HTML: ordered and unordered. Ordered lists use numbers as list items, while unordered lists use bullets.

Forms

Forms are used for user input. The <form> element wraps around interactive controls for submitting information to a web server.

Tables

Tables are used to display data in a tabular format. A table is defined with the <table> tag.
